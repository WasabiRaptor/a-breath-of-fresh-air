=========
Title       : "B_EVENT_SEPARAT"

TrackNo     Main Instrument         Details
-------     ---------------         ---------------------------------------
0           -                       Reserved special track

3          (0x05)Strings            Intro Chords + Bass strings
4          (0x07)FiddleStr          Intro Chords + backup main strings later on.
5          (0x06)Strings            Intro Chords + Back Main Strings on second loop. 
6          (0x06)Strings            Intro Chords + Another Backup string..
7          (0x23)Choir              Choir Track
8          (0x49)Strings/Fiddle     Play part of the main string melody, then only choir.
            +(0x23) Choir
9          (0x33)Flute/Piccolo      Plays the main flute + fiddle for the main track once.
            +(0x49)Fiddle/String
10         (0x48)Fiddle             Plays the main string Melody

            +(0x08)Bass
